# タスク完了時のチェックリスト

## Web プロジェクトでの作業完了時

### 必須チェック
1. **リント・フォーマット実行**
   ```bash
   cd web
   yarn lint
   ```
   - ESLintチェック実行
   - Prettierによる自動フォーマット
   - インポート文の自動整理

2. **ビルド確認**
   ```bash
   yarn build
   ```
   - TypeScriptコンパイルエラーがないか確認
   - Viteビルドが正常に完了するか確認

3. **開発サーバーでの動作確認**
   ```bash
   yarn dev
   ```
   - 実装した機能が正常に動作するか確認
   - ブラウザのコンソールにエラーがないか確認

## Crawler プロジェクトでの作業完了時

### 必須チェック
1. **リント・フォーマット実行**
   ```bash
   cd crawler
   yarn lint
   ```
   - Prettierによる自動フォーマット
   - インポート文の自動整理

2. **TypeScript コンパイル確認**
   ```bash
   npx tsc --noEmit
   ```
   - TypeScriptの型エラーがないか確認

3. **実行テスト**
   ```bash
   yarn start
   ```
   - クローラーが正常に実行されるか確認
   - データ収集が期待通りに動作するか確認

## 共通チェック項目

### Git関連
- コミット前に変更内容を確認
- 適切なコミットメッセージを記述
- 不要なファイル（.env, node_modules等）がコミットされていないか確認

### 環境変数
- .env.exampleファイルが更新されているか
- 新しい環境変数が追加された場合、ドキュメントも更新

### セキュリティ
- APIキーやシークレットがコードに直接書かれていないか
- 環境変数を適切に使用しているか
